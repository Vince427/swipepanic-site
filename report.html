Signaler un contenu

Le moyen recommandé est le bouton de signalement dans l'app : il supprime le pack localement et prépare un rapport complet.

Vous pouvez aussi écrire à report@swipepanic.app avec le nom du pack, son ID si disponible et les cartes concernées.

Les signalements sont traités manuellement.
