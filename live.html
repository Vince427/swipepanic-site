Voici le jeu

Swipe Panic est un party game offline-first : pas de compte, pas de cloud. Tu swipes des cartes, tu votes, tu obtiens un récap privé. Cinq minutes pour révéler ton groupe.

Comment le LIVE se connecte à l'appli

Le streamer affiche une carte

Une carte Swipe Panic apparaît en overlay OBS sur le stream. Tout le monde voit la même situation.

Les viewers votent dans le chat

🟢 pour Green Flag, 🔴 pour Red Flag — directement dans les commentaires TikTok.

L'appli te montre comment des milliers ont voté

Les stats de consensus apparaissent dans le récap : comment ton pays a voté, et ce que pensent les autres.

Note : le consensus est une photo périodique, pas un résultat en temps réel.

3 modes de jeu

Green Flags / Red Flags

Swipe red flags et green flags sur des situations de dating, puis découvre le profil de la partie.

Quiz

Vrai ou faux, 20 à max cartes, avec chrono de récap. Parfait pour pimenter une soirée.

Action/Vérité

Action ou Vérité, packs amis, votes locaux et récap clair en fin de partie.

3 façons de jouer

Solo

Lance une partie en solo pour découvrir le jeu à ton rythme.

Téléphone partagé (Pass & Play)

Un seul téléphone, plusieurs joueurs : on se passe le téléphone, chaque joueur vote en cachette.

Wi-Fi local (sans Internet)

Chacun joue sur son propre téléphone, connectés au même réseau local. Aucun compte requis.

Des cartes officielles, pas du hasard

Les cartes officielles sont relues et validées par l'équipe — elles ne viennent pas d'inconnus sur Internet. Les packs créés par les joueurs restent privés dans leur groupe : personne d'autre ne les voit ni ne les reçoit.
